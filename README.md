# Food Ordering POS Backend

This is a backend service for a Food Ordering POS system built using **FastAPI** and **SQLAlchemy**.

The project supports item management, item variants, and audit logging for changes.

---

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn
